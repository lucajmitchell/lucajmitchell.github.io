DevSync is a simple and easy-to-use web application designed to help manage agile development projects. It allows users to plan their work in epics, stories, tasks, and sprints, monitor progress through story points, and visualise task dependencies with automatically generated flowcharts.

Projects

Users can create and join projects which they can subsequently access and switch between at any time.

Progress

Data about the progress being made is collated visually on the dashboard. The average rate of story point completion is calculated and analysed.
