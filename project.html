DevSync is an and easy-to-use web application to help manage agile development projects. Users can plan their work in epics, stories, tasks, and sprints, monitor progress via story points, and visualise task dependencies with generated flowcharts.

Login

Users can create and login to their account.

Projects page

Users can create and join projects and switch between them.

Team dashboard

Progress data is collated in a dashboard. Average rate of story point completion is calculated and analysed.
